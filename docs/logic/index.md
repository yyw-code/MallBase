# 业务逻辑文档

业务逻辑文档说明 MallBase 中关键业务模块的规则、状态流和计算口径。

## 文档列表

| 文档 | 说明 |
|------|------|
| [member.md](./member.md) | 会员成长值、等级匹配、手动等级、会员折扣和订单快照 |
| [points.md](./points.md) | 积分账户、赠送、冻结、释放、回收、抵扣和积分兑换 |
| [wallet.md](./wallet.md) | 余额账户、流水、余额支付和充值调整 |
| [client-diy.md](./client-diy.md) | 装修方案、系统默认、启用策略、主题和客户端读取 |

## 推荐阅读方式

| 目标 | 阅读顺序 |
|------|----------|
| 理解营销权益 | 先看会员，再看积分 |
| 理解账户资金 | 先看余额，再看支付和售后相关文档 |
| 理解前台入口为何显示或隐藏 | 先看客户端装修，再看会员、积分、余额开关 |
