# 积分模块总览

积分模块负责用户积分账户、消费返积分、订单积分抵扣、积分商城兑换、积分流水和兑换单履约。

## 适用人群

| 角色 | 推荐阅读 |
|------|----------|
| 运营 / 后台管理员 | [../operation/points.md](../operation/points.md) |
| 产品 / 测试 | [../logic/points.md](../logic/points.md) |
| 开发 / 二次开发 | [../development/points.md](../development/points.md) |

## 功能范围

| 能力 | 说明 |
|------|------|
| 积分账户 | 每个用户一套积分账户，包含可用积分、冻结积分、欠账积分 |
| 积分流水 | 所有积分变动都写流水，便于排查和对账 |
| 消费返积分 | 订单完成后按规则生成奖励积分，先冻结再释放 |
| 订单积分抵扣 | 确认订单页可用积分抵扣普通订单金额 |
| 商品积分规则 | 商品和 SKU 可覆盖全局返积分规则 |
| 积分商城 | 用户可用积分兑换普通商品 SKU |
| 兑换单履约 | 后台可发货、完成、关闭兑换单 |
| 售后联动 | 退款会回收赠送积分，并按比例返还已抵扣积分 |

## 主要后台入口

| 入口 | 作用 |
|------|------|
| 设置中心 -> 积分配置 | 积分总开关、赠送、冻结、抵扣比例 |
| 积分 -> 积分规则 | 消费返积分等规则 |
| 积分 -> 积分商品 | 积分商城兑换商品 |
| 积分 -> 兑换订单 | 积分兑换单履约 |
| 积分 -> 积分流水 | 积分账户流水查询 |
| 用户 -> 用户管理 | 用户积分记录和后台积分调整 |
| 商品 -> 商品管理 | 商品 / SKU 积分赠送模式 |

## 推荐配置流程

1. 在积分配置中开启积分功能。
2. 配置消费返积分规则。
3. 配置积分冻结天数。
4. 按需开启积分抵扣，并设置抵扣比例和上限。
5. 在商品编辑页确认商品 / SKU 积分赠送模式。
6. 新增积分商品，并设置积分价、库存、限兑数量。
7. 用测试用户完成普通订单，验证赠送、冻结和释放。
8. 用测试用户完成积分兑换，验证积分扣减、库存扣减和兑换单履约。
