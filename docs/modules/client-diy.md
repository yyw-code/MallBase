# 客户端装修模块总览

客户端装修模块负责 UniApp/H5 的页面库、首页装修、个人中心装修、底部导航、悬浮按钮、主题方案和客户端运行时配置读取。

## 适用人群

| 角色 | 推荐阅读 |
|------|----------|
| 运营 / 后台管理员 | [../operation/client-diy.md](../operation/client-diy.md) |
| 产品 / 测试 | [../logic/client-diy.md](../logic/client-diy.md) |
| 开发 / 二次开发 | [../development/client-diy.md](../development/client-diy.md) |

## 功能范围

| 能力 | 说明 |
|------|------|
| 页面库 | 统一维护 UniApp 页面路径，供装修跳转选择 |
| 首页装修 | 配置搜索、轮播、导航、标题、商品分组等模块 |
| 个人中心装修 | 配置用户信息、会员、订单、余额、积分、服务菜单等模块 |
| 底部导航 | 配置原生或自定义 TabBar 的文字、图标和路径 |
| 悬浮按钮 | 配置客服、购物车、首页等全局快捷入口 |
| 主题方案 | 配置浅色、深色、自定义主题和用户选择策略 |
| 方案快照 | 每次启用装修方案时写入快照，便于追踪发布历史 |
| 运行时读取 | 客户端通过接口读取当前启用方案，缺失时回退系统默认方案 |

## 主要后台入口

| 入口 | 作用 |
|------|------|
| 客户端装修 -> 页面库 | 管理可跳转页面，导入 UniApp pages.json |
| 客户端装修 -> 首页装修 | 配置首页模块和商品来源 |
| 客户端装修 -> 个人中心装修 | 配置个人中心模块和业务入口 |
| 客户端装修 -> 底部导航 | 配置 TabBar 模式、图标和路径 |
| 客户端装修 -> 悬浮按钮 | 配置全局悬浮入口 |
| 客户端装修 -> 主题设置 | 配置主题列表、发布状态和主题策略 |

## 主要前台入口

| 页面 | 路径 |
|------|------|
| 首页 | `/pages/index/index` |
| 个人中心 | `/pages/profile/index` |
| 自定义底部导航 | `frontend/uniapp/components/mb-custom-tabbar/` |
| 装修渲染组件 | `frontend/uniapp/components/mb-decorate-renderer/` |
| 主题选择 | `/pages-sub/user/theme` |

## 推荐配置流程

1. 先确认页面库中已有要跳转的页面。
2. 不直接修改系统默认方案，先复制系统默认方案。
3. 编辑复制后的方案，调整模块、图片、跳转和样式。
4. 在后台预览确认基础布局。
5. 启用方案。
6. 打开 H5/UniApp 实际页面，确认接口返回和渲染结果一致。
7. 如果配置主题，先发布主题，再配置主题策略。
