# 会员模块总览

会员模块负责会员开关、会员等级、成长值、会员折扣、规格会员价、用户会员账户和订单会员优惠快照。

| 产品 / 测试 | [../logic/member.md](../logic/member.md) |
| 开发 / 二次开发 | [../development/member.md](../development/member.md) |

## 功能范围

| 能力 | 说明 |
|------|------|
| 会员总开关 | 控制客户端会员入口、订单会员权益和商品会员配置展示 |
| 会员等级 | 按成长值门槛配置等级名称和等级折扣 |
| 成长值 | 订单完成后按实付金额累计成长值 |
| 自动升级 | 成长值达到更高门槛后自动升级，不自动降级 |
| 手动等级 | 后台可为用户手动设置等级，并支持长期或限时锁定 |
| 等级折扣 | 商品可参与会员等级折扣 |
| 规格会员价 | 商品可按 SKU 配置独立会员价 |
| 订单快照 | 下单时保存会员等级和优惠金额，历史订单不回算 |

## 主要后台入口

| 入口 | 作用 |
|------|------|
| 会员 -> 会员配置 | 会员总开关、成长值比例 |
| 会员 -> 会员等级 | 等级名称、成长值门槛、折扣、状态 |
| 用户 -> 用户管理 | 查看用户会员信息、手动设置用户会员等级 |
| 商品 -> 商品管理 | 配置商品会员权益模式和 SKU 会员价 |
| 订单 -> 订单详情 | 查看会员优惠和成长值快照 |

## 推荐配置流程

1. 在会员配置中开启会员功能。
2. 设置成长值比例，例如每实付 1 元累计 1 成长值。
3. 配置会员等级，至少保留一个成长值门槛为 `0` 的启用等级。
4. 在商品编辑页确认会员权益模式：全局、关闭、等级折扣或规格会员价。
5. 用测试用户下单，确认订单试算页出现会员优惠。
6. 完成订单后，确认用户成长值增加，并按规则自动升级。
7. 如需人工保级，在用户管理中手动设置会员等级并填写原因。
