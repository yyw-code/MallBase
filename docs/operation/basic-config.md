# 基础配置操作文档

本页汇总常见基础配置入口，说明配置项影响哪些业务链路，以及保存后应该怎么验证。具体菜单名称以后台实际路由和权限同步结果为准。

## 上传配置

常见配置：

| 配置 | 说明 |
|------|------|
| 默认上传驱动 | 本地存储、OSS、COS |
| 图片大小限制 | 单图、多图上传大小和数量 |
| 文件大小限制 | 单文件、多文件上传大小和数量 |
| 视频大小限制 | 单视频、多视频上传大小和数量 |
| 证书大小限制 | 微信支付证书等敏感文件上传限制 |

验证建议：

1. 后台上传一张图片。
2. 在商品、装修或文章中引用该图片。
3. 打开前台页面，确认图片可访问。
4. 如果切换云存储，确认历史素材 URL 仍可访问或已完成迁移。

安全注意：上传配置涉及密钥时，不要把 SecretId、SecretKey、AccessKey、证书内容写进文档、截图或提交记录。

## 支付配置

常见配置：

| 配置 | 影响 |
|------|------|
| 微信支付状态 | 客户端是否展示微信支付和充值方式 |
| 余额支付状态 | 客户端订单支付是否可选择余额支付 |
| 微信支付 V3 参数 | 微信支付预下单、回调、退款 |

验证建议：

1. 修改支付开关后，打开客户端支付方式接口或订单支付页。
2. 用测试订单验证支付方式是否展示。
3. 余额支付需确认用户余额充足。
4. 微信支付需确认商户号、APIv3 密钥、证书和回调配置。

## 积分配置

常见配置：

| 配置 | 影响 |
|------|------|
| 启用积分功能 | 客户端积分入口、新发放、新抵扣、新兑换 |
| 启用积分赠送 | 订单完成后是否产生新的奖励积分 |
| 积分冻结天数 | 奖励积分多久从冻结转可用 |
| 启用积分抵扣 | 确认订单页是否可用积分抵扣 |
| 积分抵扣比例 | 多少积分抵扣 1 元 |
| 积分抵扣上限比例 | 单笔订单最多抵扣比例 |

详细操作见 [points.md](./points.md)。

## 会员配置

常见配置：

| 配置 | 影响 |
|------|------|
| 启用会员功能 | 客户端会员入口、新订单会员权益、成长值累计 |
| 成长值比例 | 每实付 1 元累计多少成长值 |

详细操作见 [member.md](./member.md)。

## 客户端配置

常见配置：

| 配置 | 影响 |
|------|------|
| 客户端品牌 | H5/UniApp 展示名称、Logo、分享信息 |
| 协议内容 | 用户协议、隐私政策、关于我们 |
| 客服信息 | 个人中心或服务菜单展示 |
| 主题策略 | 用户是否可自选主题、后台默认主题模式 |

装修方案、页面库和主题的详细操作见 [client-diy.md](./client-diy.md)。

## 配置变更后的检查

修改基础配置后，建议按下面顺序确认：

1. 后台保存后刷新页面，确认配置值已回显。
2. 打开对应客户端页面，确认接口返回和页面展示一致。
3. 涉及支付、积分、会员、余额时，用测试用户走完整链路。
4. 涉及装修、主题或图片时，同时检查后台预览和实际 H5/UniApp。
5. 涉及静态资源、H5 或 Admin 发布时，确认构建产物和 Nginx 路径正确。

## 常见注意事项

- 开关类配置通常只影响后续业务动作，不回溯修改历史订单、历史快照或历史流水。
- 余额、积分、会员等级这类账户数据不要直接改数据库，优先通过后台操作或服务方法写入流水。
- 支付、上传、短信等密钥不应出现在公开文档、提交记录、截图或日志中。
- 修改配置后如果前台不生效，先检查接口返回，再检查前端缓存和当前启用的装修方案。
