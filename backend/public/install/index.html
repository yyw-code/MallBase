    2. 环境检测

      3. 数据库配置

      连接目标：按部署方式选择，也可以手动填写宿主机 IP。
      安装要求：目标数据库必须为空；如果数据库不存在，当前账号需要能创建该库并写入表结构。

      4. Redis 配置

      连接目标：按部署方式选择，也可以手动填写宿主机 IP。
      安装要求：请选择一个空的 Redis DB；如果当前 DB 已有键，请改用其他空 DB 后再继续。

      5. 管理员账号

      用作后台入口、Client API 基址、本地上传访问域名。安装后可在「系统设置 → 系统配置 → 基础」修改。
      高级选项
