# 系统默认装修素材

本目录存放 MallBase 装修系统的必要默认素材，由仓库直接 commit。

这些素材会被后台装修编辑器、后台手机预览、UniApp / H5 客户端共同使用，不属于演示数据。
全新安装即使不导入演示数据，也必须保留本目录文件。

## 边界

- `static/decorate/`：系统默认装修素材，安装必需；包含装修画布图片与悬浮按钮系统图标。
- `static/demo/`：演示商品、演示评价、演示充值套餐等演示数据素材，仅导入演示数据时需要。

## 数据库引用

`backend/install/data/schema/13_mb_upload_asset.sql` 会注册本目录素材为 `static` driver 的系统素材。
默认装修方案通过素材 ID 引用这些记录，后台和客户端通过素材解析后的 `full_url` 展示。
