# 客户端配置默认静态资源

本目录存放 mall-base 客户端配置域的默认素材。后台会展示和维护这些配置，运行端会读取这些配置；本目录不是“客户端专属资源”边界。

| 文件 | 用途 | 推荐尺寸 | 格式 |
|---|---|---|---|
| logo.png | 客户端主图标（页面顶栏、我的页、分享模块） | 512×512 | PNG 透明 |
| launch.png | Uniapp 编译为 App 时的启动屏 | 1080×2340 | JPG/PNG |
| share-cover.png | H5/微信小程序"分享给好友"时封面 | 500×400 | PNG/JPG |

首页轮播图、悬浮按钮图标等装修系统默认素材在 `static/decorate/`。

## 替换方式

登录后台 → 系统设置 → 客户端配置，上传新图片即可覆盖默认。
