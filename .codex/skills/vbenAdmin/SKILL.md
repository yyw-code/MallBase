# Vben Admin Skills 索引（MallBase）

前端规则按子目录拆分，请按场景选用：

1. `backend-driven-routing`：路由由后端菜单驱动
2. `upload-component-first`：上传必须优先复用 Upload 组件
3. `iconpicker-standard`：图标选择统一使用 IconPicker 规范
4. `modal-form-layout`：Modal 表单布局一致性
5. `admin-theme-consistency`：后台主题（明暗色）一致性规范
6. `api-path-param`：前端 API 与后端 `/:id` 路径参数一致
7. `upload-field-normalize`：上传字段回填与提交规范
8. `e2e-webantd-realapi`：web-antd 对接真实后端的 E2E 测试规范
9. `format-after-change`：web-antd 代码改动后必须做局部格式化
