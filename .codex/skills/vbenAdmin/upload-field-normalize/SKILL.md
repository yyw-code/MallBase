# Vben 规则：上传字段回填与提交规范

## 适用范围

`frontend/admin/apps/web-antd/src/views/**` 中使用 `Upload` 的编辑页。

## 强制规则

1. 编辑回填时，上传字段使用对象结构：`{ url, full_url, name }`。
2. 展示优先读取后端返回的完整地址字段（`full_url/*_full_url`）。
3. 提交时只回传存储路径字段（`url`），不回传 `full_url`。
4. 批量修改逻辑必须覆盖图片字段（不能只改价格库存）。

## 自检清单

- [ ] 主图、轮播图、SKU 图在编辑页都能回显。
- [ ] 保存后二次进入编辑，图片仍能稳定回显。
- [ ] 不依赖前端硬编码域名拼接。
